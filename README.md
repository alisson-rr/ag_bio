# Agenda Magnética - Linktree

Site estilo linktree para a Agenda Magnética, seguindo a identidade visual da marca.

## 🎨 Identidade Visual

- **Cores principais:**
  - Verde petróleo: #2C7464
  - Rosa suave: #FEA5A4
  - Preto: #292726
  - Creme: #F5F0EB

- **Tipografia:**
  - Red Hat Display (títulos)
  - Montserrat (textos)

## 🚀 Como executar

### Instalar dependências
```bash
npm install
```

### Executar em desenvolvimento
```bash
npm run dev
```

### Build para produção
```bash
npm run build
```

### Preview da build
```bash
npm run preview
```
